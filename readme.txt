This plugin allows you to add contact information in a widget and display it with or without icons.


== Description ==

ContaGen Widget plugin shows your contact info with or without icons. Add the widget to a widget position like the footer or sidebar, fill in the info, choose icon option, alignment and press save. Your contact info with icons is now being displayed on the front-end of your website!

You can add your contact information to the sidebar. You can choose between 2 types of formats: plain text or with icons.

The plugin add classes to the div tags, so you can create your own styles in your stylesheet

**Features**

This plugin features possibility to add:

* Title
* Address
* Phone
* Email


== Installation ==

Via WordPress -

1. Install as a regular WordPress plugin.
2. From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'contagen-widget.zip' file and upload it. Or
3. Upload faq plugin to the /wp-content/plugins/ directory
4. Activate the plugin through the 'Plugins' menu in WordPress


Via FTP -

1. Upload .zip file to your WordPress plugin directory and unzip it.
2. Go your Plugins setting via WordPress Dashboard and activate it.


== Frequently Asked Questions ==

**The icons are not showing right?**

The widget adapts to the themes widget position.

**The plugin is not working on my website how come?**

1. Check if it's enabled.
2. Check if you run the latest version of Wordpress
